# Todo App #

# Technologies Used
- Java 8
- Spring Boot
- String MVC
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven
- Docker
- JUnit and Mockito


# Requirement for Development and Build
JDK 1.8
Maven 3.x

### Build and Run ###
```
$ cd /path/to/project-root
$ mvn package
$ java -jar target/iyzico-todo.jar
```

#### In addition ####
You can use docker-compose to setting up infrastructure and running.
